## Standard Input
Its the third type of [Data Streams](/obisdian_ntoes/scriptss/Data Streams.md)
U have to store the value of **read** into [Variables](/obisdian_ntoes/scriptss/Variables.md) 
>[!example]-
>echo "Enter ure name"
>**read** myname
>echo "Your name is: $myname"
